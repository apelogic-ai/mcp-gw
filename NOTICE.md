# Notices

This project wraps the Google Workspace CLI (`@googleworkspace/cli`) and uses Google Discovery
metadata to generate part of the MCP tool catalog.

The repository includes generated catalog data derived from:

- the pinned `@googleworkspace/cli` package version recorded in this repository;
- public Google Workspace Discovery documents.

Some early catalog design was informed by public Google Workspace MCP wrapper projects. If future
changes copy third-party source or catalog data directly, preserve the applicable license notices
here.
